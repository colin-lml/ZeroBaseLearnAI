# 导数

导数是微积分中的核心概念之一，函数在某一点的导数就是该函数所代表的曲线在这一点上的切线斜率（不清楚的话，建议找相应教程）， 常用的导数公式有

**常数函数**

若 f(x) = C（C 为常数），则f'(x) = 0

**幂函数**

$若 (f(x) = x^n)（n 为常数），则f'(x) = n *x^{n-1}$

示例 $f(x)=x^2, 则f'(x)=2x$

**指数函数(暂时用不上)**
$自然指数函数：若 f(x) = e^x，则f'(x) = e^x$
$一般指数函数：若 f(x) = a^x，则f'(x)= a^x * ln(a)$
